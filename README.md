# react-native-spotify-apr

library for use spotify sdk in React Native app

## Installation

```sh
npm install react-native-spotify-apr
```

## Usage

```js
import SpotifyApr from "react-native-spotify-apr";

// ...

const result = await SpotifyApr.multiply(3, 7);
```
